Flow Management API (1.0.0)

Download OpenAPI specification:Download

The Flow Management API enables seamless management of various components such as projects, storage, flows, functions, prompts, executor images, retrievers, and secrets. It provides endpoints for creating, updating, retrieving, and deleting these resources, ensuring efficient flow execution and orchestration.
